# 一、镜头景别（必做预设）
- 大特写（Extreme Close-up）：眼部、唇部、细节情绪
- 特写（Close-up）：面部表情、情绪核心
- 中近景（Medium Close-up）：胸像、对话常用
- 中景（Medium Shot）：半身、动作、关系
- 中全景（Medium Wide）：全身+部分环境
- 全景（Wide Shot）：人物全身，环境清晰
- 大全景（Extreme Wide）：环境叙事、氛围、史诗感

# 二、运镜语言（最核心！AI视频必用）
- 固定镜头（Static）：稳定、沉稳
- 推镜（Dolly In）：渐进、强调、压迫
- 拉镜（Dolly Out）：抽离、揭示环境
- 跟镜（Tracking）：跟随人物移动
- 摇镜（Pan）：水平扫视
- 仰镜（Low Angle）：威严、力量、压迫
- 俯镜（High Angle）：渺小、孤独、上帝视角
- 环绕运镜（Orbit）：电影感、旋转、高级
- 缓慢平移（Slow Pan）：舒缓、文艺
- 手持晃动（Handheld）：纪实、紧张、真实
- 轻微抖动（Subtle Shake）：紧张感、悬疑
- 镜头呼吸（Breathing Lens）：柔和沉浸
- 变焦冲击（Zoom Burst）：冲击、强烈情绪

# 三、构图（电影感灵魂）
- 中心构图：稳定、正式、主角感
- 三分构图：自然、舒适、电影标准
- 对称构图：庄严、美学、仪式感
- 留白构图：意境、高级、极简
- 框架构图（Frame within Frame）：电影感极强
- 引导线构图：透视、纵深、叙事
- 低饱和构图：克制、高级
- 紧凑构图：压迫、紧张

# 四、光影（决定质感）
- 软光（Soft Light）：柔和、自然、皮肤细腻
- 硬光（Hard Light）：锐利、戏剧、强烈
- 逆光（Backlight）：轮廓、唯美、高级
- 侧光（Side Light）：立体、情绪、电影感
- 顶光（Top Light）：严肃、压抑、悬疑
- 伦勃朗光（Rembrandt）：经典肖像、高级
- 蝴蝶光（Butterfly）：柔美、人像质感
- 剪影（Silhouette）：意境、极简、力量
- 氛围光（Atmospheric Light）：朦胧、雾感
- 丁达尔光（God Ray）：史诗、神圣
- 冷暖对比光：冲突、戏剧、高级感
- 暗调光影（Low Key）：深沉、悬疑
- 亮调光影（High Key）：明亮、治愈

# 五、景深与画质
- 浅景深（Shallow DOF）：突出主体、电影虚化
- 深景深（Deep DOF）：环境清晰、叙事
- 焦外虚化（Bokeh）：唯美、柔和
- 锐利对焦（Sharp Focus）：清晰、质感
- 柔焦（Soft Focus）：梦幻、朦胧
- 电影颗粒（Film Grain）：复古、质感
- 镜头眩光（Lens Flare）：自然、真实
- 暗角（Vignette）：聚焦主体、电影感
