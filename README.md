# SketchShot

SketchShot is an AI Storyboard Canvas for creators.

当前仓库的正式产品名为 `SketchShot`，定位是“无限画布上的故事板创作工具”。

当前说明：

- 对外产品名已统一为 `SketchShot`
- 技术兼容前缀如 `WXHB_*`、`wxhb-*` 目前暂时保留，用于兼容现有部署、Docker 镜像和环境变量
- 后续如果需要再做一轮彻底的技术前缀迁移，应单独评估兼容影响后再处理
