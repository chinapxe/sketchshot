# 协作行为准则

更新日期：2026-04-08

SketchShot 希望把仓库维护成一个对新手友好、对专业协作者也高效的协作空间。无论你是提 Issue、发 PR、补文档、讨论方案，还是只是指出一个命名不清楚的小问题，都欢迎参与。

## 我们鼓励的行为

- 用事实、上下文和复现信息推动讨论。
- 尊重不同经验水平、语言表达习惯和技术背景。
- 针对问题本身给出反馈，避免把分歧上升到个人评价。
- 在指出不足时尽量补充证据、替代方案或最小修复思路。
- 保护隐私、商业信息和第三方版权内容，不在公开讨论中泄露敏感信息。

## 不可接受的行为

- 人身攻击、羞辱、威胁、歧视性表达或持续骚扰。
- 恶意刷屏、挑衅、故意制造无效争论。
- 公开分享他人的隐私数据、真实密钥、客户资料或未经授权的素材。
- 明知会破坏协作氛围，仍反复进行不尊重他人的互动。

## 维护者的责任

- 维护者有权编辑、隐藏、锁定或移除明显不符合本准则的内容。
- 对影响协作安全和秩序的行为，维护者可以限制互动权限。
- 在处理争议时，维护者会尽量基于上下文、证据和仓库目标做判断。

## 报告与处理

- 如果问题适合公开沟通，可在 Issue、PR 或 Discussion 中说明。
- 如果内容涉及隐私、安全、骚扰或不适合公开披露，请不要公开贴出细节，等待仓库补充私下联系渠道后再提交。
- 在正式公开发布前，维护者仍需补充更明确的私下联系入口。
