# Support

更新日期：2026-04-08

## 适合使用哪种渠道

- 使用 Issue：
  - 可稳定复现的 Bug
  - 明确的功能建议
  - 文档错误或缺失

- 使用 Pull Request：
  - 你已经完成了代码或文档修改，并愿意直接贡献

- 使用 Security 渠道：
  - 凭据泄露
  - 安全漏洞
  - 不适合公开披露的问题

## 提问前建议先检查

- [README.md](./README.md)
- [CONTRIBUTING.md](./CONTRIBUTING.md)
- [ROADMAP.md](./ROADMAP.md)
- [docs/project-structure.md](./docs/project-structure.md)
- [docs/open-source-checklist.md](./docs/open-source-checklist.md)

## 反馈时请尽量提供

- 复现步骤
- 环境信息
- 日志、截图或录屏
- 期望结果与实际结果

请不要在公开反馈中贴出真实 API Key、客户数据、隐私信息或未经授权的素材。
